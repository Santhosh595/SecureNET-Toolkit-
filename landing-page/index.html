Why SecureNET

Enterprise-grade security, zero cost

Built with proven cryptographic libraries and production-ready patterns. No dependencies on cloud services or paid APIs.

AES-256 Encryption

Military-grade symmetric encryption via Fernet. Keys derived from passwords using PBKDF2 with 480,000 iterations.

Integrity Verification

SHA-256 hash comparison on every decryption. Detects tampering before you open compromised files.

Intrusion Detection

Rule-based IDS detecting port scans and packet floods. Configurable thresholds with alert cooldown.

Live Dashboard

Real-time Flask web dashboard showing captured packets, alerts, and network statistics.

SQLite Storage

All packets and alerts persisted to SQLite with indexed columns for fast querying and analysis.

Clean Architecture

Modular Python codebase with separated concerns: crypto, database, detection, and presentation layers.

Toolkit

Two tools, one mission

Each tool addresses a different aspect of cybersecurity: data-at-rest protection and network-level threat detection.

FileGuard

AES-256 + SHA-256 File Encryption

Encrypt and decrypt files with AES-256 symmetric encryption. Keys are derived from your password using PBKDF2. Every decryption verifies file integrity against a stored SHA-256 hash.

Python Tkinter cryptography hashlib
  • Password-based key derivation (PBKDF2, 480K iterations)
  • AES-256 encryption via Fernet
  • SHA-256 integrity verification on decrypt
  • Desktop GUI with Tkinter
  • Secure key storage with salt

Network Sniffer

Real-Time IDS with Flask Dashboard

Capture live network packets, store them in SQLite, and detect intrusion patterns. A Flask web dashboard shows packets and alerts in real time.

PortMap

Multi-threaded Port Scanner & Risk Analyzer

Scan any host for open ports using raw sockets. Identifies running services and flags high-risk exposures with actionable risk notes. No Nmap required.

Python Sockets Threading Rich
  • Raw socket scanning, no Nmap dependency
  • 4 scan profiles: quick / common / full / custom
  • Service identification on 17+ known ports
  • Risk levels: LOW / MEDIUM / HIGH per port
  • Live progress + JSON export

HashDetect

Hash Identifier & Wordlist Cracker

Paste any hash and instantly identify its type with confidence scoring. Supports 18+ hash formats. Optional wordlist-based cracking for weak hashes like MD5 and SHA-1.

Python Flask hashlib Rich
  • 18+ hash types detected (MD5 to Argon2)
  • Confidence scoring: HIGH / MEDIUM / LOW
  • Algorithm category tagging (WEAK / CRYPTOGRAPHIC / PASSWORD)
  • Optional wordlist crack with custom list support
  • CLI + web dashboard + JSON export
